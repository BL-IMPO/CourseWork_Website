Getting Started

How to Analyze Text

Learn the basic steps to analyze your text using ReadScore:

  1. Navigate to the Analyze page
  2. Choose between pasting text or uploading a file
  3. For text input: Paste your content in the text area
  4. For file upload: Select your document (TXT, DOC, DOCX, PDF)
  5. Click "Analyze Text" or "Analyze File"
  6. Review your comprehensive readability report

Readability Formulas Explained

Flesch Reading Ease

Measures how easy a text is to read. Scores range from 0-100, with higher scores indicating easier reading.

  • 90-100: Very Easy (5th grade)
  • 80-89: Easy (6th grade)
  • 70-79: Fairly Easy (7th grade)
  • 60-69: Standard (8th-9th grade)
  • 50-59: Fairly Difficult (10th-12th grade)
  • 30-49: Difficult (College level)
  • 0-29: Very Difficult (College graduate)

Dale-Chall Readability

Based on vocabulary familiarity and sentence length. Uses a list of 3,000 familiar words.

SMOG Index

Stands for "Simple Measure of Gobbledygook." Estimates the years of education needed to understand a text.

Other Formulas

We also calculate Automated Readability Index, Coleman-Liau Index, and Gunning Fog Index for comprehensive analysis.

Technical Details

Linguistic Features Analyzed

  • Word count and sentence count
  • Average sentence length
  • Average word length
  • Vocabulary richness (unique word ratio)
  • Syllables per word
  • Part-of-speech distribution (nouns, verbs, adjectives)
  • Complex conjunction frequency
  • Punctuation density

AI Model

Our neural network model is trained on diverse text samples and uses 16 linguistic features to predict text complexity with high accuracy.

Best Practices

Text Length

For most accurate results, use texts with at least 100 words. Very short texts may not provide reliable readability scores.

Language Support

Currently optimized for English text. Performance may vary with other languages.

File Formats

Supported formats: TXT, DOC, DOCX, PDF. Maximum file size: 10MB.
